🎬 Netflix Data Dashboard
🔍 Project Overview

This project presents an interactive Netflix data dashboard that visualizes trends in content distribution, growth, genres, countries, and ratings. It transforms raw Netflix dataset insights into a visually engaging interface, allowing users to explore how the platform’s content library has evolved over time.

🎯 Objectives
Visualize the distribution of Movies vs TV Shows
Analyze how content additions have evolved yearly
Identify the most popular genres and countries
Explore content ratings distribution
Build an interactive dashboard for intuitive data exploration
🛠️ Tools & Technologies
Python (data preprocessing)
Pandas (data cleaning & transformation)
Matplotlib & Seaborn (initial analysis)
HTML, CSS, JavaScript (interactive dashboard development)
🧹 Data Preparation
Removed records with missing essential values (type, title)
Handled missing data:
country → filled as "Unknown"
rating → filled as "Not Rated"
Converted date_added to datetime format
Engineered new features:
year_added for trend analysis
📊 Dashboard Highlights
📌 Content Type Distribution
Interactive donut chart showing Movies vs TV Shows
Movies form the majority of Netflix’s catalog
📌 Content Added Over Time
Timeline visualization of yearly additions
Rapid growth observed after 2015, peaking around 2019–2020
📌 Top Genres
Horizontal bar chart of top 10 genres
Dominated by:
International Movies
Dramas
Comedies
📌 Top Countries
Shows leading content-producing countries
United States, India, and the United Kingdom lead the platform
📌 Content Ratings
Grid view of ratings distribution
TV-MA and TV-14 are the most common
📌 Release Year Trends
Visualizes how content production spans from 2007 to 2021
📈 Key Insights
Netflix has experienced rapid expansion in content after 2015
Movies significantly outnumber TV Shows
Content strategy emphasizes international and diverse genres
A few genres dominate, indicating focused audience targeting
The platform hosts content from a wide range of countries, highlighting global reach
📌 Conclusion

This dashboard provides a comprehensive visual analysis of Netflix’s content ecosystem. It highlights the platform’s growth, genre preferences, and global expansion, offering insights into how Netflix has scaled its content strategy over time while catering to diverse audiences.
